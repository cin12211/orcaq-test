# Implementation Plan: Connection Environment Tags

**Branch**: `013-connection-env-tags` | **Date**: 2026-04-11 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/013-connection-env-tags/spec.md`

## Summary

Add environment tagging to database connections. Users can assign up to 3 colored, named tags to each connection to signal environment context (prod, uat, test, dev, local). A built-in **strict mode** prevents accidental connections to critical environments by requiring the user to type the exact phrase `"this is production"` before connecting. Tags are managed globally in a new Settings panel. The implementation follows the existing dual-platform persistence pattern (localforage/IDB for web, Electron IPC for desktop) with a new `environment-tags` collection and a connection schema migration.

---

## Technical Context

**Language/Version**: TypeScript 5.x  
**Primary Dependencies**: Nuxt 3 (SPA), Vue 3 Composition API, Pinia, VeeValidate v4 + Zod, shadcn-vue, Tailwind CSS 4, localforage, Electron IPC  
**Storage**: Dual-platform: localforage → IndexedDB (web) | Electron IPC → native store (desktop). New collection `environment-tags` added to both.  
**Testing**: Vitest (unit — hooks + utils), Playwright (e2e — connection form flows)  
**Target Platform**: Web browser (SPA) + Electron desktop (dual-target, same codebase)  
**Project Type**: Desktop/web application (database management tool)  
**Performance Goals**: Tag picker renders without perceptible delay (<50ms); strict mode dialog appears immediately on connect click  
**Constraints**: Max 3 tags per connection enforced at UI + schema validation; tag names unique case-insensitively; offline-capable (IDB + Electron store, no server)  
**Scale/Scope**: ~5–50 tags in library; tag count per user session is trivial

| Tag management UI     | New "Environment Tags" panel in existing Settings modal                      | research.md §D6 |
| Max-3 enforcement     | Picker UI (disabled state) + Zod schema (`array().max(3)`)                   | research.md §D7 |
