# 🎉 What's New in v1.1.1

We are excited to introduce the **Database Tools** module, a powerful addition to your workflow that simplifies database maintenance and schema management.

## 🚀 New Features

### Database Tools

A dedicated module for advanced database operations, now accessible from the sidebar.

- **Backup & Restore:** Easily create full database backups and restore them when needed. Manage your data snapshots with confidence.
- **Schema Diff:** Compare schemas between different databases or environments. Visualize structural differences in tables, views, and functions with an intuitive diff tree.
